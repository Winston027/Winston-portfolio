# Winston Wen Studio

一个为内容设计师 Winston Wen 打造的个人作品集网站，整体风格参考了新粗野主义视觉。

网站特征：

- **大胆排版**：大字号标题、强烈黑色描边和块面、少量但有个性的色彩。
- **聚焦内容设计**：作品、思考与个人介绍都围绕“内容体验设计”展开。
- **纯前端静态站点**：直接双击打开 `index.html` 即可查看，也可以使用本地静态服务器。

## 使用方式

### 直接本地打开

1. 在 Finder 中打开项目文件夹。
2. 双击 `index.html`，用浏览器（推荐 Chrome / Safari）打开。

### 使用本地静态服务器（可选，需要 Node.js 环境）

如果你已经安装了 Node.js，可以在项目根目录运行本地静态服务器，避免浏览器的本地文件安全限制：

```bash
cd /Users/shidawen/Desktop/个人作品集网站
npm install serve --save-dev
npm run start
```

然后在浏览器中访问终端输出的本地地址（一般是 `http://localhost:3000` 或类似端口）。

## 结构说明

- `index.html`：主页面结构，包含各个区块（首页、作品、内容与思考、关于、联系）。
- `styles.css`：全部样式文件，包含新粗野主义风格的排版和布局。
- `script.js`：简单的页面内平滑滚动逻辑。
- `package.json`：用于管理本地开发时的静态服务器依赖（没有强制前端框架依赖）。

## 自定义内容

- 将 `index.html` 中作品、文案、联系方式等替换成你真实的项目和信息即可。
- 如需新增板块，可以复制现有 `section` 结构并复用对应的样式类。
